WWM Tradutor PT-BR

Instalador automático com interface gráfica moderna. Instala e atualiza a tradução com um clique. ~40MB

  • Suporte a Steam, Epic e Standalone
  • Detecção automática da plataforma
  • Instalação com 1 clique
  • Backup e restauração de originais
  • Solicita admin quando necessário
  • Suporte a escalas de DPI

Faça a instalação manual se preferir ter controle total sobre os arquivos.

  • translate_words_map_en (~30k)
  • translate_words_map_en_diff (~20k)
  • Sem dependências
  • Portátil

Instalação

Siga os passos abaixo para instalar a tradução

1

Baixe o Launcher

Clique no botão de download acima para baixar o instalador

2

Execute

Abra o arquivo WWM_Tradutor_PTBR.exe

3

Selecione o Jogo

Clique em "Selecionar" e escolha o wwm.exe do seu jogo (Steam, Epic ou Standalone)

4

Instale e Jogue!

Clique em Instalar e depois inicie o jogo pelo launcher

Instalação Manual

Se preferir instalar manualmente, baixe o arquivo ZIP e extraia os arquivos para a pasta de locale do seu jogo:

Steam/Epic: ...\Where Winds Meet\Package\HD\oversea\locale\
Standalone: ...\wwm_standard\LocalData\Patch\HD\oversea\locale\

Substitua os arquivos translate_words_map_en e translate_words_map_en_diff

Como usar o Launcher

Passo a Passo

  1. 1 Baixe e execute o WWM_Tradutor_PTBR.exe (não precisa instalar)
  2. 2 Clique em "Selecionar" e encontre o arquivo wwm.exe do seu jogo (veja como abaixo)
  3. 3 Clique em "Instalar Tradução" e aguarde o download
  4. 4 Pronto! Clique em "Jogar" para iniciar o jogo traduzido

Como encontrar o wwm.exe

Steam
Método rápido: Biblioteca Steam → Clique direito no jogo → Gerenciar → Navegar arquivos locais

Caminho: Abra a pasta do jogo, depois navegue até:
📁 Engine → 📁 Binaries → 📁 Win64r → 📄 wwm.exe

Caminho completo:
C:\Program Files (x86)\Steam\steamapps\common\Where Winds Meet\Engine\Binaries\Win64r\wwm.exe
Epic Games
Método rápido: Epic Games → Biblioteca → Clique nos 3 pontos (...) → Gerenciar → Abrir pasta de instalação

Caminho: Abra a pasta do jogo, depois navegue até:
📁 Engine → 📁 Binaries → 📁 Win64r → 📄 wwm.exe

Caminho completo:
C:\Program Files\Epic Games\WhereWindsMeet\Engine\Binaries\Win64r\wwm.exe
Standalone (Standard ou Lite)
Localização padrão: O jogo standalone é instalado na pasta que você escolheu

Caminho: Abra a pasta wwm_standard (ou wwm_lite), depois navegue até:
📁 Engine → 📁 Binaries → 📁 Win64r → 📄 wwm.exe

Caminho comum:
C:\Program Files\wwm\wwm_standard\Engine\Binaries\Win64r\wwm.exe
C:\Program Files\wwm\wwm_lite\Engine\Binaries\Win64r\wwm.exe

💡 Dica: O launcher detecta automaticamente a plataforma e cria backup dos arquivos originais. Você pode restaurar a qualquer momento!

Instalação Manual

Passo a Passo

  1. 1 Baixe o arquivo traducao_ptbr.zip
  2. 2 Extraia o conteúdo do ZIP (dois arquivos)
  3. 3 Navegue até a pasta de tradução do seu jogo (veja como abaixo)
  4. 4 Cole os arquivos e substitua quando perguntado

Arquivos incluídos no ZIP

Arquivos de Tradução
• translate_words_map_en (~30.000 strings)
• translate_words_map_en_diff (~20.000 strings)

Caminho completo:
C:\Program Files (x86)\Steam\steamapps\common\Where Winds Meet\Package\HD\oversea\locale\
Epic Games

Caminho: Abra a pasta do jogo, depois navegue até:
📁 Package → 📁 HD → 📁 oversea → 📁 locale

Caminho completo:

Standalone (Standard ou Lite)
Localização padrão: O jogo standalone é instalado na pasta que você escolheu

Caminho: Abra a pasta wwm_standard (ou wwm_lite), depois navegue até:
📁 LocalData → 📁 Patch → 📁 HD → 📁 oversea → 📁 locale
